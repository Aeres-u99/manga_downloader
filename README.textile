h1. Manga Downloader

Manga Downloader is a Windows/Linux python script that does what it says for mangavolume.com
Downloads into .cbz format, can optionally download into .zip instead.
Untested on Mac OS.

h2. Usage

h3. Mandatory:

*-e {ending chapter}*

*-n {name of manga}*

*-s {starting chapter}*

h3. Optional:

*-d {download path}*
Default behavior is to use current working directory.

*-o*
Overwrites downloaded chapters if it already exists.  Default behavior is to skip over it.

*-z*
Downloads into .zip format instead of .cbz

e.g. ./manga.py -n 'Air Gear' -s 1 -e 100 -d ~/Manga

The name of the manga can be enclosed in either single or double quotes.  Quotes are optional if manga name is only one word or if escaping spaces with a backslash.

h2. Bugs

* Does not error check for invalid starting/ending chapters.
* .zip images are packaged inside a temp directory
